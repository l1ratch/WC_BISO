### **Definitions of Terms**

#### **Brute force**  
A hacking method that involves trying all possible character combinations (e.g., passwords) until the correct one is found.  

#### **Cryptanalysis**  
The science of breaking ciphers and cryptographic systems by analyzing their weaknesses.  

#### **Classification**  
In machine learning, the task of assigning objects to predefined categories (classes).  

#### **Clustering**  
In machine learning, grouping objects into clusters based on similarity without predefined categories.  

#### **Defacement**  
Hacking a website to alter its content (often for visual or political purposes).  

#### **Input units**  
Neurons in a neural network that receive raw data for processing.  

#### **Output units**  
Neurons in a neural network that return the final computation result.  

#### **IoT (Internet of Things)**  
A network of physical devices (e.g., smart gadgets) connected to the internet and exchanging data.  

#### **Keylogger**  
Malicious software or hardware that records keystrokes to steal passwords and sensitive data.  

#### **Miner**  
A program or device that performs cryptocurrency mining using computational resources.  

#### **Phishing**  
A fraudulent attack where a hacker impersonates a trusted entity to steal sensitive data (e.g., passwords).  

#### **Piggybacking**  
1) Unauthorized use of someone else's internet connection.  
2) In cybersecurity, covert data transmission within a legitimate connection.  

#### **Salami shaving**  
A fraud method where small amounts are deducted from multiple accounts, often going unnoticed.  

#### **Signature-based scanning**  
An antivirus scanning method that compares files against known malware signatures.  

#### **Heuristic scanning**  
An antivirus scanning method that detects suspicious program behavior, even if signatures are unknown.  

#### **Symmetric encryption**  
Encryption where the same key is used for both encryption and decryption.  

#### **Asymmetric encryption**  
Encryption using a key pair: a public key (for encryption) and a private key (for decryption).  

#### **Trigger**  
A condition or event that activates malicious code execution (e.g., a specific date or user action).  

#### **Trojan**  
Malicious software disguised as legitimate software to gain unauthorized system access.
